# wikipedia_search_app
A minimalist yet powerful interface that transforms raw curiosity into instant knowledge. Built with pure HTML, CSS, and JavaScript, this application taps directly into Wikipedia’s public API to deliver lightning-fast, context-rich search results.

# Experience Highlights
Real-time Wikipedia search with instant feedback
Clean, distraction-free interface for focused discovery
Responsive design across devices
Dynamic result rendering with concise summaries
Direct links to full Wikipedia articles

# Tech Stack
HTML5 – Semantic structure
CSS3 – Modern styling & layout
JavaScript – API integration & interactivity
Wikipedia REST API – Live content source
